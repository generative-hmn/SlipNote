# SlipNote 사용 설명서

SlipNote는 빠른 메모 작성과 정리를 위한 macOS 메뉴바 앱입니다.

## 빠른 입력

전역 단축키로 어디서든 빠르게 메모를 작성할 수 있습니다.

- **Cmd+Shift+N** - 빠른 입력창 열기
- 메모 입력 후 **Cmd+숫자(0-9)** - 해당 카테고리에 저장
- **Esc** - 입력창 닫기 (내용은 캐시됨)
- **Cmd+A** - 전체 텍스트 선택
- **Cmd+V/C/X/Z** - 붙여넣기/복사/잘라내기/실행취소

## 브라우저

전역 단축키로 브라우저 창을 열어 메모를 관리합니다.

- **Cmd+Shift+B** - 브라우저 열기/닫기

### 카테고리 필터

- **Cmd+숫자(0-9)** - 해당 카테고리 필터 토글
- **Cmd+T** - 휴지통 필터 토글
- **Cmd+위/아래** - 카테고리 간 이동
- **Esc** - 필터 해제 및 검색어 초기화

### 슬립 탐색

- **위/아래 화살표** - 슬립 선택 이동
- **Enter** - 선택한 슬립 상세보기
- **Cmd+Enter** - 선택한 슬립 편집 모드로 열기
- **더블클릭** - 슬립 상세보기

### 슬립 선택

- **클릭** - 단일 선택
- **Cmd+클릭** - 다중 선택 토글
- **Shift+클릭** - 범위 선택
- **Esc** - 다중 선택 해제

### 슬립 작업

- **Option+숫자(0-9)** - 선택한 슬립의 카테고리 변경
- **Cmd+Delete** - 선택한 슬립을 휴지통으로 이동
- **Cmd+C** - 선택한 슬립을 마크다운으로 클립보드에 복사
- **Cmd+P** - 슬립 고정/해제 (고정된 슬립은 상단에 표시)
- **드래그 앤 드롭** - 슬립을 카테고리로 이동

### 검색

- **Cmd+F** - 검색창 포커스

### 상세보기

- **Cmd+Enter** - 편집 모드 시작
- **Esc** - 뒤로가기 (편집 중이 아닐 때)
- **좌/우 화살표** - 버전 히스토리 탐색

### 기타

- **Cmd+N** - 새 슬립 만들기
- **Cmd+,** - 설정 열기

## 카테고리

기본 카테고리:
- **0** Inbox - 기본 저장 위치
- **1** Idea
- **2** Plan
- **3** Task
- **4** Event
- **5** Journal
- **6** Library
- **7** Reference
- **8** Archive
- **9** Temp

설정에서 카테고리 이름과 색상을 변경할 수 있습니다.

## 슬립 고정 (Pin)

중요한 슬립을 상단에 고정할 수 있습니다.

- **Cmd+P** - 선택한 슬립 고정/해제
- **우클릭 > Pin to Top / Unpin** - 고정/해제
- 고정된 슬립은 📌 아이콘과 함께 목록 상단에 표시됩니다.

## 내보내기

- 카테고리 우클릭 > **Export to Markdown** - 카테고리 전체를 마크다운으로 내보내기
- **Cmd+C** - 선택한 슬립을 마크다운으로 복사

## 자동 백업

설정 > Data 탭에서 자동 백업을 설정할 수 있습니다.

- **Off** - 자동 백업 비활성화
- **Daily** - 매일 백업
- **Weekly** - 매주 백업
- **Monthly** - 매월 백업

백업 파일은 `~/Library/Application Support/SlipNote/Backups/` 폴더에 저장되며, 최근 10개의 백업이 유지됩니다.

## Spotlight 검색

SlipNote의 모든 슬립은 macOS Spotlight에서 검색할 수 있습니다.

- **Cmd+Space** 로 Spotlight를 열고 슬립 내용을 검색
- 검색 결과를 클릭하면 SlipNote 브라우저에서 해당 슬립이 열립니다.

## URL Scheme

외부 앱이나 자동화 도구에서 SlipNote를 제어할 수 있습니다.

```
# 새 슬립 생성
slipnote://new?content=메모내용&category=1

# 브라우저 열기
slipnote://browse
slipnote://browse?category=3

# 검색
slipnote://search?query=키워드

# 입력창 열기
slipnote://input
```

**활용 예시:**
- macOS Shortcuts 앱에서 빠른 메모 생성
- Alfred/Raycast 워크플로우 연동
- 스크립트를 통한 자동 메모 생성

## 데이터

- 설정 > Data 탭에서 데이터 파일 위치 확인 및 백업 가능
- 모든 데이터는 로컬에 저장됩니다
